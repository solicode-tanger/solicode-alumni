# solicode-alumni
Cette plateforme fédère les diplômés de Solicode pour stimuler l'échange d'expertises et créer des synergies professionnelles durables entre les talents du digital. Elle accompagne chaque lauréat dans son évolution de carrière tout en consolidant l'impact de notre communauté sur le marché du travail national et international.
